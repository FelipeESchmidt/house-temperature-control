MÁQUINA VIRTUAL E DOCKER

    - Criar uma pasta em uma máquina com docker instalado.
    - Copiar os seguintes arquivos para dentro desta nova pasta.
        - app/ (pasta app completa)
        - Dockerfile
        - docker-compose.yml
    - Rodar a aplicação de subscribe em PHP
        - docker-compose up -d
    - Acessar aplicação
        - curl http://localhost:80

    *(
        - Parar aplicação
            - docker-compose stop
            - docker-compose down
    )*
